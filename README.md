# What is Markdown?

**Markdown** is a plain text formatting syntax for writers. It allows you to quickly write structured content for the web, and have it seamlessly converted to clean, structured HTML.

## Basic Markdown Formatting

### Headings

```
# Heading 1
## Heading 2
### Heading 3
```

### Text

```
*italic*
**bold**
***bold-italic***
```
